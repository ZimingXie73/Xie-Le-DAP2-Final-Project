# Fire Perimeter Analysis

This project Analyze Environmental Regulation and Firm Dynamics Under China's SO2 Emission Reduction Policy (2005-2010).

## Setup

```bash
conda env create -f environment.yml
```

## Usage

1. Run "Part 1" to clean data:
   ```bash
   code/Final Project Part 1.qmd
   ```

2. Run "Part 2" for data exploration:
   ```bash
   code/Final Project Part 2.qmd
   ```
  
3. Run "Part 3" for data visualization:
   ```bash
   code/Final Project Part 3.qmd
   ```

3. Run "app.py" for Dashboard:
   ```bash
   code/app.py
   ```

4. Output are rendered into 3 parts: Part 1 as .html; Part 2 & 3 as .pdf

5. See our Policy Writeup @ Policy Writeup.pdf; Presentation Slides @ DAP2_Presentation_Xie_Sun.pdf
